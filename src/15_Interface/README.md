# 接口

## 介绍
    接口描述了一组抽象的规范。
    基本接口： 只包含方法集（一组方法的机会）的接口。
    通用接口： 只要包含类型集（一组类型的集合）的接口。

## 基本接口

### 声明
    type Person interface {
        Say(string) string
        Walk(int)
    }

### 实现
    实现： 根据规范提供具体的功能
    面向接口编程：内部实现改变而功能不变，不影响之前的使用，可以随意替换


## 空接口
    type Any interface {
    }
    内部没有方法集，所有类型的方法集都是空集的超集，所以Any接口可以保存任何类型的值。
